üìù How to Use in Webex Contact Center

1. Build the Component:

  • Run npm install to install dependencies
  • Run npm run build to create production bundle
  • The compiled file will be in dist/agent-desktop-component.js

2. Host the JavaScript file:

  • Upload the built file to a web server (must be HTTPS)
  • Ensure the file is accessible via a public URL

3. Configure in Agent Desktop Layout:

  • Go to Webex Contact Center Administration Portal
  • Navigate to Desktop Layout configuration
  • Add a new Custom Component
  • Select "Web Component" as the component type
  • Enter the component tag: <agent-desktop-widget></agent-desktop-widget>
  • Enter the script URL: https://your-server.com/agent-desktop-component.js

4. Development Mode:

  • Run npm run dev for development with auto-reload
  • Run npm run serve to start development server on port 3000
